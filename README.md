# Usage
// Selfhost Version
// GUI Version
// WebApp

**Selfhost**
```
1. Clone App
git clone https://github.com/multitools-ap-mvp/pdf-editor-app.git

2. Install
npm install

2.5.
Put your source PDF in /input/ Directory

3. Extract data from your PDF
npm run extract

4. Generate editable HTML + PDF
npm run generate

# 5. (Optional) Start web editor
npm run edit
```
